# 画面構成

## 画面一覧

| 画面 ID | 画面名 | 目的 | 対応ストーリー |
|---|---|---|---|
| SCR-1 | ホーム | データ状態の確認と主要機能への導線 | S-1, S-2, S-3 |
| SCR-2 | データ取り込み | Zaim / Moneytree エクスポートの取り込み | S-1, S-2 |
| SCR-3 | AI チャット | 自然言語での家計相談と応答表示 | S-3, S-4, S-5, S-6, S-7 |
| SCR-4 | 設定 | データ削除とプライバシー管理 | S-8 |

## ナビゲーションマップ

| 起点 | 操作 | 遷移先 | 備考 |
|---|---|---|---|
| SCR-1 | 「データを取り込む」 | SCR-2 | Empty 状態の主導線 |
| SCR-1 | 「AI に相談する」 | SCR-3 | データ取り込み済みが前提 |
| SCR-1 | 「設定」 | SCR-4 | フッターナビ |
| SCR-2 | 取り込み成功 | SCR-1 | Success 状態 |
| SCR-2 | 戻る | SCR-1 | |
| SCR-3 | 戻る / ホーム | SCR-1 | |
| SCR-4 | 削除確認後 | SCR-1 | Empty 状態に戻る |
| SCR-4 | 戻る | SCR-1 | |

## 画面ごとのコンポーネントツリー

### SCR-1 ホーム

- ヘッダー（アプリ名）
- メイン
  - データ状態サマリー（件数・最終取り込み日）
  - プライマリ CTA（取り込み / 相談）
  - クイック質問例
- フィードバック領域（Empty / Success メッセージ）
- フッターナビ（ホーム / 取り込み / 相談 / 設定）

### SCR-2 データ取り込み

- ヘッダー（戻る、タイトル）
- メイン
  - アプリ選択（Zaim / Moneytree）
  - ファイル選択
  - 取り込みボタン
- フィードバック領域（Loading / Error / Validation Error / Success）

### SCR-3 AI チャット

- ヘッダー（戻る、タイトル）
- メイン
  - 会話履歴（ユーザー質問 / AI 応答）
  - 応答セクション（金額・明細・改善案・良かった点）
- 入力エリア（テキスト入力、送信ボタン）
- フィードバック領域（Loading / Error / Empty）

### SCR-4 設定

- ヘッダー（戻る、タイトル）
- メイン
  - データ保存の説明（端末内のみ）
  - 削除ボタン（破壊的操作）
  - 削除確認ダイアログ
- フィードバック領域（Success / Error）

## 共通 UI 要素

- フッターナビ（4タブ）
- ステータスメッセージ領域
- プライマリボタン / セカンダリボタン

## レスポンシブ方針

- Mobile（320px〜）: 単一カラム、フッターナビ固定、入力は画面下部
- Tablet / Desktop: MVP では Mobile 優先。広い画面では最大幅 480px 中央配置
