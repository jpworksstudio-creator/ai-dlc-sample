# ストーリー

## ストーリー S-1: Zaim エクスポートの取り込み

父として、Zaim のエクスポートファイルを Web アプリに取り込みたい。家計データを AI 分析の入力にできるようにするため。

### 受け入れ基準

- Given 父がスマホブラウザで Web アプリを開いている
  When Zaim のエクスポートファイルを選択して取り込む
  Then 取引データがブラウザローカルに保存され、取り込み成功が表示される

### トレーサビリティ

要件: FR-1, FR-3, NFR-1, NFR-5

### UI の要否

UI 必須

## ストーリー S-3: 自然言語での AI 質問

父として、自然言語で AI エージェントに家計について質問したい。手動で明細を確認する負担を減らすため。

### 受け入れ基準

- Given 取り込み済みの家計データがある
  When 父が「先月の総支出はいくら？」など自然言語で質問を送信する
  Then 30秒以内に AI からテキスト応答が返る

### トレーサビリティ

要件: FR-4, FR-5, NFR-1, NFR-2

### UI の要否

UI 必須

## ストーリー S-4: カテゴリ別支出の把握

父として、食費・日用品・固定費などカテゴリ別の支出額を AI に聞きたい。誤分類を補正した正確な金額を把握するため。

### 受け入れ基準

- Given 取り込み済みの家計データがあり、父が手動補正後の正解金額を用意している
  When 父が「先月の食費はいくら？」と質問する
  Then AI は食費・日用品・固定費について、手動補正後の金額と±10%以内の回答を返す

### トレーサビリティ

要件: FR-6, NFR-3

### UI の要否

UI 必須

## ストーリー S-5: 取引明細の理解

## ストーリー S-6: 無駄遣いの指摘と改善案
