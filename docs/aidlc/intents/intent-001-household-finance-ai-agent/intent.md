# インテント

## インテント ID

`intent-001-household-finance-ai-agent`

## 元のリクエスト

個人の家計簿AIエージェント。Zaim、Moneytree 等の家計簿アプリから情報をエクスポートし、支出の正しい情報を取得。正しい収支情報を基に、AIエージェントがお金の使い方の改善案や良かった点を評価する。

## 概要

4人家族（父・母・2歳の子）の**父**向け Web 家計 AI エージェント MVP。Zaim / Moneytree のエクスポートを端末内に取り込み、カテゴリ誤りや明細の判別困難さを補正した上で、自然言語で支出把握・無駄遣い指摘・改善提案・良かった点の評価を受けられる体験を検証する。

## スラッグ

`household-finance-ai-agent`

## 種別

プロトタイプ

## 分類

グリーンフィールド（新規）

## プロダクト仮説の承認

状態: 承認済み

## スコープ

### 対象範囲

- Zaim / Moneytree からのエクスポートデータ取り込み
- 端末内（ブラウザローカル）でのデータ保存
- 支出明細の補正・カテゴリ整理（仮説として検証）
- AI エージェントへの自然言語での相談
- 無駄遣い指摘・改善案・良かった点の評価（セットで提供）
- スマホブラウザ向け Web UI

### 対象外

- ネイティブアプリ
- 家計簿アプリとの API 自動連携
- 複数世帯・複数ユーザー管理
- 投資・資産運用アドバイス
- レシート OCR / 自動仕訳
- 構築（Construction）・コード生成・インフラ構築（本 Harness の対象外）
