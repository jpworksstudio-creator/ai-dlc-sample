# Construction Exit Checklist

## 確認項目

- [ ] Construction Input Readiness が承認済み
- [ ] Construction Plan が承認済み
- [ ] 全 Unit の必須 Review が承認済み
- [ ] 全 Layer の Human Checkpoint が完了
- [ ] Final Integration Report が承認済み
- [ ] Traceability が Requirement / Story / Screen / Unit / Code Summary まで接続済み
- [ ] Remaining Issues が許容済みまたは Deferred
- [ ] Operations に進まないことを確認した

## 未解決なら停止する条件

- 未承認 Review がある
- 未完了 Unit がある
- Final Integration が未承認である
- Operations に進もうとしている
