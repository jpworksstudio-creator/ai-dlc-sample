# Units Checklist

## 確認項目

- [ ] 単一 Unit で十分か、複数 Unit が必要かが判断されている
- [ ] 単一 Unit の場合、その理由が記録されている
- [ ] 複数 Unit の場合、Unit 間依存が記録されている
- [ ] Story と Unit の対応表がある
- [ ] 各 Unit が提供するユーザー価値がある
- [ ] Construction に渡せる完了条件がある
- [ ] 独立して検証できるユーザー価値を確認した
- [ ] Feature Area を確認した
- [ ] Domain Boundary を確認した
- [ ] デプロイ単位を確認した
- [ ] チーム所有を確認した
- [ ] 運用責任を確認した

## 未解決なら停止する条件

- Units の判断が未承認である
- Story と Unit の対応がない
- 複数 Unit の依存が不明である
- Construction に渡せる完了条件がない
