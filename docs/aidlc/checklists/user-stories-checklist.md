# User Stories Checklist

## 確認項目

- [ ] Persona が domain-specific である
- [ ] 各 Story に `S-*` ID がある
- [ ] User Story が価値を説明している
- [ ] Acceptance Criteria が Given / When / Then 形式である
- [ ] 各 Story が FR / NFR に trace されている
- [ ] UI が必要な Story と不要な Story が分かれている
- [ ] 重複する Story がない
- [ ] 要件にない機能を Story として追加していない

## 未解決なら停止する条件

- Stories が未承認である
- FR/NFR との trace がない
- UI Story が不明で Wireframe に進めない
- Acceptance Criteria が検証不能である
