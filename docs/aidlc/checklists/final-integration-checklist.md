# Final Integration Checklist

## 確認項目

- [ ] 全 Unit が完了している
- [ ] Lint を実行した
- [ ] Type Check を実行した
- [ ] Unit Test を実行した
- [ ] Integration Test を実行した
- [ ] Build を実行した
- [ ] 必要な場合のみ E2E を実行した
- [ ] 必要な場合のみ UI Review の最終確認を実行した
- [ ] `final-integration-report.md` がある

## 未解決なら停止する条件

- Final Integration が未承認である
- 必須 check が失敗している
- 未完了 Unit がある
