# Build and Test Checklist

## 確認項目

- [ ] Lint を実行した、または実行できない理由がある
- [ ] Type Check を実行した、または実行できない理由がある
- [ ] Unit Test を実行した、または実行できない理由がある
- [ ] Build を実行した、または実行できない理由がある
- [ ] 失敗がある場合、修正方針が記録されている
- [ ] Layer / Unit の report が保存されている

## 未解決なら停止する条件

- 失敗した check を無視して次へ進もうとしている
- Report が未承認である
