# Requirements Checklist

## 確認項目

- [ ] Intent Summary がある
- [ ] Functional Requirements が `FR-*` で採番されている
- [ ] Non-Functional Requirements が `NFR-*` で採番されている
- [ ] 要件が可能な限り合格 / 不合格で判断できる
- [ ] Facts / Hypotheses / Assumptions / Decisions / Open Questions が分離されている
- [ ] Success Metrics が Product Hypothesis と対応している
- [ ] Out of Scope が明示されている
- [ ] 不明点が最大 5 件以内で質問されている

## 未解決なら停止する条件

- Requirements が未承認である
- FR/NFR が検証不能である
- 重要な Open Question が未解決である
- Product Hypothesis と対応しない要件が混ざっている
