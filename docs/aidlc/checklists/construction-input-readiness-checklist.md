# Construction Input Readiness Checklist

## 確認項目

- [ ] Components がある
- [ ] Responsibilities がある
- [ ] Dependencies がある
- [ ] Services がある
- [ ] Cross-cutting Concerns がある
- [ ] Data Model がある
- [ ] API Overview がある
- [ ] External Dependencies が確認されている
- [ ] Unit ID がある
- [ ] Unit と Story の対応がある
- [ ] Unit と Requirement の対応がある
- [ ] Unit のユーザー価値がある
- [ ] Unit 間依存がある、または不要理由がある
- [ ] 完了条件がある

- 不足情報を AI が推測で補完しそうな状態である
- Unit への traceability が不十分である
- Inception へ差し戻すべき gap がある
