# Workflow Checklist

## 確認項目

- [ ] 実行する Skill が明示されている
- [ ] 省略する Skill と理由が明示されている
- [ ] 条件付き Skill と実行条件が明示されている
- [ ] 実行順序が Product Discovery から Inception Exit までになっている
- [ ] Construction が含まれていない
- [ ] 各工程に人間の承認ゲートがある
- [ ] 未承認の Product Hypothesis を Inception に混ぜていない

## 未解決なら停止する条件

- Workflow が未承認である
- Construction や code generation が含まれている
- 承認なしで次工程へ進む手順になっている
