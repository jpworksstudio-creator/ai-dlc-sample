# UI Review Checklist

## 確認項目

- [ ] UI が approved stories と対応している
- [ ] UI が approved wireframes と対応している
- [ ] Loading / Empty / Error / Success / Validation Error / Permission Denied が確認されている
- [ ] Mobile 幅で確認されている
- [ ] 入力欄とボタンの accessible name / label が確認されている
- [ ] Focus と keyboard 操作が確認されている
- [ ] Error message が具体的である
- [ ] Output Encoding を確認した

## 未解決なら停止する条件

- UI Review が未承認である
- UI が Story または Wireframe とずれている
- Blocking accessibility issue がある
