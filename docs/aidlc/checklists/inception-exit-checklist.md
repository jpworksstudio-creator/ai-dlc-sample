# Inception Exit Checklist

## 確認項目

- [ ] Product Hypothesis が承認済みである
- [ ] Workflow が承認済みである
- [ ] Requirements が承認済みである
- [ ] Stories が承認済みである
- [ ] Wireframes が承認済みである
- [ ] Application Design が承認済みである
- [ ] Units の判断が承認済みである
- [ ] Traceability Matrix が Hypothesis から Unit of Work までつながっている
- [ ] Remaining Issues が Construction 前に許容できる状態である
- [ ] Construction に進むかどうかは人間が次に明示する必要がある

## 未解決なら停止する条件

- 未承認の工程がある
- 重大な Open Question が残っている
- Traceability に説明なしの未接続がある
- Construction 開始の明示依頼がない
