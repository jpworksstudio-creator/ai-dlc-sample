# Product Discovery Checklist

## 確認項目

- [ ] 対象ユーザーが明確である
- [ ] 現在の行動や業務が説明されている
- [ ] 困っている場面が具体的である
- [ ] 現在の代替手段が記録されている
- [ ] 理想的な体験が説明されている
- [ ] 主要なユーザージャーニーがある
- [ ] 最も危険な仮説が明示されている
- [ ] MVP で検証する成功指標が測定可能である
- [ ] MVP では対象外にする体験が明示されている
- [ ] Facts / Hypotheses / Decisions / Open Questions が分離されている

## 未解決なら停止する条件

- Product Hypothesis が未承認である
- 仮説と事実が混在している
- 成功指標がない、または検証不能である
- 対象外が不明で MVP スコープが広がり続ける
